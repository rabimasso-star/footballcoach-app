\# Football Coach App ⚽



A platform for football coaches to plan trainings, manage teams and build better training sessions.



---



\## Project structure



footballcoach-app

│

├─ backend   → NestJS API

├─ mobile    → React Native / Expo app

└─ web       → Next.js web app



---



\## Planned features



\- Training planner

\- Exercise library

\- Team management

\- Session builder

\- Player tracking

\- AI-generated training sessions



---



\## Tech stack



Backend

\- NestJS

\- Prisma

\- PostgreSQL



Web

\- Next.js

\- React

\- TypeScript



Mobile

\- React Native

\- Expo



---



\## Development



Start backend



cd backend  

npm install  

npm run start:dev  



Start web



cd web  

npm install  

npm run dev  



Start mobile



cd mobile  

npm install  

npm start
